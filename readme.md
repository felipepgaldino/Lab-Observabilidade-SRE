# SRE Monitoring Lab – Prometheus + Grafana + Docker

## Overview

Practical observability lab created to simulate production monitoring scenarios using SRE concepts and open-source tools.

## Architecture

* Ubuntu Linux
* Docker
* Prometheus
* Grafana
* Node Exporter
* Blackbox Exporter
* Nginx

## Monitoring Implemented

* CPU monitoring
* Memory monitoring
* HTTP endpoint validation
* Service availability check
* Alert generation

## Incident Simulation

Nginx service interruption generated:

* probe_success = 0
* target DOWN
* alert trigger

## Skills Demonstrated

* Monitoring
* Observability
* Alerting
* Troubleshooting
* Docker networking
* YAML configuration
* Incident response
